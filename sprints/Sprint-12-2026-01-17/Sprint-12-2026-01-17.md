### Completed Deliverables
- **Accountability reports**: Service (`lib/reports/accountabilityReport.ts`), API `GET /api/reports/accountability`, UI at `/dashboard/admin/org/reports/accountability`, CSV/PDF/Excel export.
- **Attendance reporting**: Derived from count cards by platoon/date; service, API `GET /api/reports/attendance`, UI with summary stats and trend chart, export. *Per-recruit attendance deferred (requires new data model).*
- **Recruit status reports**: Service with PII-masked names; API `GET /api/reports/recruit-status`; UI with status filter, rank filter, distribution chart, export.
- **Platoon/squad performance metrics**: Service aggregating count cards + recruit counts; API `GET /api/reports/performance`; dashboard with comparison table and Recharts (avg present rate, recruit count by platoon), export.
- **Export**: Shared helper `lib/reports/export.ts` (escapeCsv, rowsToCsv); all report pages support CSV, PDF (pdf-lib), Excel (xlsx).
- **Data visualization**: Recharts used in attendance (trend by date), recruit status (distribution), performance (bar charts); responsive containers.
- **Scheduled reports**: Extended `reportSchedules` with `reportType` (org | accountability | attendance | recruit_status | performance); cron `GET /api/cron/org-reports` runs all types and emails CSV; schedules UI includes report type and date-range preset for non-org types; `lastError` stored on schedule doc on failure.
- **Report scope**: `lib/reports/scope.ts` — access via existing view permissions (`view_company`, `view_series`, `view_own_platoon`, `manage_organizations`); data scoped by `getOrganizationalScopeForUser`.

## Sprint Goal
Create comprehensive reporting and analytics system including accountability reports, attendance tracking and reporting, recruit status reports, platoon/squad performance metrics, export functionality, data visualization, and scheduled report generation.

## Overview
This sprint focuses on building a complete reporting and analytics system for CountCard, providing administrators and users with tools to generate reports, track attendance, analyze recruit status, measure platoon/squad performance, visualize data, and schedule automated reports.
