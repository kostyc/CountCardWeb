# CountCard Expo fonts

**SpaceMono-Regular.ttf** — dev/debug monospace (loaded in `app/_layout.tsx`).

**Colossalis** (optional) — add `Colossalis.ttf` here, register in `app/_layout.tsx` via `useFonts`, and set `fontFamilies.headingNative` in `@countcard/ui/tokens` to `'Colossalis'`.

Until Colossalis is bundled, headings use **Georgia** (`fontFamilies.headingNative`).
